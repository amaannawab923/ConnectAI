## Installation

### Prerequisites
- Python 3.11+
- pip or poetry

### Using a Connector

```bash
# Clone the repository
git clone https://github.com/amaannawab923/ConnectAI.git
cd ConnectAI

# Install a connector (e.g., Notion)
cd source-notion
pip install -r requirements.txt

# Use the connector
python
>>> from src.connector import NotionConnector
>>> from src.config import NotionConfig
>>>
>>> config = NotionConfig(token="ntn_your_token_here")
>>> connector = NotionConnector(config)
>>>
>>> # Discover available streams
>>> streams = await connector.discover()
>>> print(streams)
>>>
>>> # Read data
>>> records = []
>>> async for record in connector.read(stream="pages"):
>>>     records.append(record)
```

## Lessons Learned

Building this system taught us valuable lessons about AI agent systems:

### 1. AI Agents Hallucinate
**Problem**: TesterAgent reported "150 tests passed ‚úÖ" but code couldn't import.

**Solution**: Built validation layer to actually run tests before trusting agent claims.

### 2. Static Knowledge Goes Stale
**Problem**: Generated code used old token format (`secret_*` instead of `ntn_*`).

**Solution**: Added guardrails to treat API tokens as opaque strings, never hardcode formats.

### 3. State Management is Critical
**Problem**: Early failures in one agent blocked later successes.

**Solution**: Clear error states when tests pass. Multi-agent systems need careful state cleanup.

**Read more**: See [Day 2 Post](https://github.com/amaannawab923/ConnectAI/blob/master/DAY_2_POST.md) for detailed analysis.
